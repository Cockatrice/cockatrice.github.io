Search Syntax Help

The search bar recognizes a set of special commands similar to some other card databases.
In this list of examples below, each entry has an explanation and can be clicked to test the query. Note that all searches are case insensitive.

Name:
birds of paradise (Any card name containing the words birds, of, and paradise)

"birds of paradise" (Any card name containing the exact phrase "birds of paradise")

Rules Text (Oracle):
o:flying (Any card text that has the word flying)
o:"first strike" (Any card text that has the exact phrase "first strike")

o:"{T}" o:"add one mana of any color" (Any card text that has a tap symbol and the phrase "add one mana of any color")

Types:
t:angel (Any card with the type angel)
t:angel t:legendary (Any angel that's also legendary)
t:basic (Any card with the type basic)

t:arcane t:instant (Any card with the types arcane and instant)

Colors:
c:w (Any card that is white)
c:wu (Any card that is white or blue)
c:wum (Any card that is white or blue, and multicolored)
c:c (Any colorless card)

ci:w (Any card that has white in it's color identity)

Power, Toughness, Mana Value:
tou:1 (Any card with a toughness of 1)
pow>=8 (Any card with a power greater than or equal to 8)

mv=7 (Any card with a mana value equal to 7)

Rarity:

r:mythic (Any card that has the mythic-rare rarity)

Format:
f:standard (Any card that can be played in standard)
banned:modern (Any card that is banned in modern)
restricted:vintage (Any card that is restricted in vintage)

legal:pauper (Any card that is legal in pauper)

Edition:
set:lea (Cards that appear in Alpha, which has the set code LEA)

e:lea or e:leb (Cards that appear in Alpha or Beta)

Negate:

c:wu -c:m (Any card that is white or blue, but not multicolored)

Branching:

t:sliver or o:changeling (Any card that is either a sliver or has changeling)

Grouping:

t:angel -(angel or c:w) (Any angel that doesn't have angel in its name and isn't white)

Regular Expression:
/^fell/ (Any card name that begins with "fell")
o:/counter target .* spell/ (Any card text with "counter target *something* spell")

o:/for each .* and\/or .*/ (/'s can be escaped with a )
