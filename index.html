How to compile your own binaries

Cockatrice is an open source, multi-platform application for playing tabletop card games over a network. The program's server design prevents users from manipulating the game for unfair advantage.
The client also provides a single-player mode, which allows users to brew while offline.

Get Started

  1. Download the app from the sidebar on the left & install it

  2. Open 'Cockatrice' and run the 'Oracle' card database tool

  3. Build a deck in client, or import one

  4. Start playtesting offline, or connect to a public server

Enjoy playing your favorite card games!
